Référence de la classe CPluginSerialPort

Cette classe se connecte en serial port si les périphériques connues ne sont pas détectés. En faite elle n'est pas très utile, sauf pour le programmeur. Attention, une seul instance de chaque plugin est utilisable, donc si deux périphérique sont reconnu, un seul s'associera avec le plugin. Plus de détails...

Description détaillée

Cette classe se connecte en serial port si les périphériques connues ne sont pas détectés. En faite elle n'est pas très utile, sauf pour le programmeur. Attention, une seul instance de chaque plugin est utilisable, donc si deux périphérique sont reconnu, un seul s'associera avec le plugin.

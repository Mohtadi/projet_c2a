Référence de la classe CPluginBluetooth

Classe du plugin bluetooth. Permet de reconnaitre les périphériques bluetooth, et d'envoyer des fichiers sur un périphériques. L'association des appareil n'est pas géré pour l'instant. Plus de détails...

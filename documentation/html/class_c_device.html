13 Décembre 2011 : création (Fabien Bradmetz) Conteneur pour les périphériques. L'application utilise une liste de CDevice ou chaque CDevice représente un périphérique à surveiller. Lorsqu'un périphérique est détecté, un plugin lui est attaché.
